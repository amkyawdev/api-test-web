# API Test Hub

A modern web application for testing and interacting with various AI service providers.

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## 🌟 Features

- **Multi-Provider Support** - Connect to 23+ AI providers
- **100+ Models** - Access to hundreds of AI models
- **Real-time Chat** - Interactive chat interface
- **Session History** - Save and load previous conversations
- **Secure Storage** - API keys stored locally in browser
- **Responsive Design** - Works on desktop and mobile

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/amkyawdev/api-test-web.git

# Navigate to project directory
cd api-test-web

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
npm run preview
```

## 🔒 Privacy

- API keys are stored **locally in your browser** only
- No data is sent to external servers (except API requests to AI providers)
- Chat history is saved in browser localStorage

## 📝 Usage

1. Select an AI service provider
2. Enter your API key
3. Test the connection
4. Start chatting!
